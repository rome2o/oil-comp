# Visual Guidance for Gamma Design

**Purpose:** Specifications for all charts, tables, and visual elements in the presentation  
**Goal:** Translate markdown content into high-impact visual designs in Gamma

## ğŸ¨ Overall Design Guidelines

### Color Palette
- **Primary (Trust/Data):** Deep blue (#1E3A8A or similar)
- **Secondary (Urgency/Loss):** Red/orange (#DC2626 for negative data)
- **Tertiary (Growth/Gain):** Green (#059669 for positive data)
- **Neutral (Background):** Light gray (#F3F4F6), white (#FFFFFF)
- **Text:** Dark gray (#1F2937) for body, black (#000000) for headings

### Typography
- **Headings:** Bold, large, sans-serif (e.g., Inter, Helvetica)
- **Body Text:** Regular, readable, sans-serif (16-18pt minimum)
- **Data Points:** Bold for numbers, regular for labels
- **Emphasis:** Use bold, color, or sizeâ€”not multiple effects

### Layout Principles
- **Hierarchy:** Most important element = largest/boldest
- **White Space:** Leave breathing room around elements
- **Alignment:** Consistent left/center alignment per slide
- **Consistency:** Same chart types use same colors throughout

## ğŸ¨ Design Tool Tips for Gamma

### Using Gamma Features

1. **Animations:**
   - Slide 1: Arrows animate in (up for traffic, down for conversions)
   - Slide 7: Dual tracks appear side-by-side, then merge with formula
   - Slide 15: Fork animates splitting into two paths

2. **Charts:**
   - Use Gamma's built-in chart tool for:
     - Slide 2 funnel
     - Slide 5 pie charts
     - Slide 9 bar chart
     - Slide 12 ROI table
   - Ensure data is editable for future updates

3. **Icons:**
   - Use Gamma's icon library or import from:
     - Heroicons (free, matches modern aesthetic)
     - Font Awesome (widely recognized)
   - Keep icon style consistent throughout

4. **Color Consistency:**
   - Set custom color palette in Gamma settings
   - Apply consistently across all slides
   - Use same red for problems, same green for solutions

5. **Typography:**
   - Heading: 36-48pt bold
   - Subheading: 24-32pt regular
   - Body: 18-20pt regular
   - Data points: 28-36pt bold
